一个 CLAUDE.md 配置文件，用于优化 Claude Code 的编程表现，灵感来源于 Andrej Karpathy 对 LLM 编程常见陷阱的观察与总结。

**核心技术**：基于提示工程（Prompt Engineering）方法论，利用 LLM 的上下文理解能力，通过结构化的指令文件引导 AI 生成更准确、更符合项目规范的代码。
